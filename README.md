# Parque Nacional Carara  

Inició como Reserva Biológica en 1978 mediante el Decreto Ejecutivo № 8491-A. Posteriormente, después de veinte años cambió su categoría de manejo a Parque Nacional a través del Decreto Ejecutivo № 27411-MINAE (SINAC, s.f).
